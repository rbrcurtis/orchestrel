# Pi AI Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Orchestrel's Claude Code / Claude Agent SDK runtime with Pi while keeping the Orc frontend ↔ orcd app protocol stable.

**Architecture:** `bin/orc` becomes a wrapper around the canonical `pi` CLI. `orcd` uses the Pi TypeScript SDK directly and becomes Pi-native internally, but it continues emitting the existing Orchestrel `stream_event` / `result` / `session_exit` / `context_usage` messages to avoid frontend protocol churn. Resource semantics move fully to Pi: user config under `~/.pi`, `AGENTS.md` for context, Pi prompt templates/commands for slash commands, and Pi skills as skills; no `.claude` compatibility layer.

**Tech Stack:** TypeScript, Vite/React, socket.io, Unix socket JSONL protocol, Pi packages (`@earendil-works/pi-coding-agent`, `@earendil-works/pi-ai`), Vitest, better-sqlite3, YAML config.

---

## Non-Negotiable Decisions

1. **No Claude compatibility layer.** Do not read `.claude/skills`, `.claude/commands`, or `CLAUDE.md` for the new runtime.
2. **Pi canonical resources.** Use Pi's default user config layout (`~/.pi/...`) and standard project `AGENTS.md` behavior.
3. **Commands and skills stay distinct.** Slash commands are Pi prompt templates / commands. Skills are Pi skills. Do not recreate Claude Code's merged skill/command semantics.
4. **`orc` CLI wraps `pi` CLI.** It should add Orchestrel provider/model convenience, then delegate to Pi.
5. **`orcd` uses Pi SDK directly.** Do not route daemon sessions through `pi --mode rpc` unless the SDK proves insufficient.
6. **Frontend protocol remains stable.** The Orc FE ↔ orcd message envelope does not need to change for this migration.
7. **orcd internals become Pi-native.** Remove Claude SDK, Claude JSONL session assumptions, Claude skill disabling, Claude executable paths, and Anthropic-only env plumbing where they are runtime implementation details.
8. **Provider/proxy simplification is in scope.** Pi can speak Anthropic and OpenAI-style formats, so old proxy-only code paths should be audited and simplified where Pi owns the capability.

```ts
function isRecord(value: unknown): value is Record<string, unknown> {
  return typeof value === 'object' && value !== null;
}

export function mapPiEventToOrcdPayload(event: unknown): unknown {
  if (!isRecord(event)) return event;

  if (event.type === 'turn_end') {
    return {
      type: 'result',
      subtype: 'success',
      message: event.message,
      toolResults: event.toolResults ?? [],
    };
  }

  return event;
}

export function getContextUsageFromPiEvent(event: unknown): { contextTokens: number; contextWindow: number } | null {
  if (!isRecord(event)) return null;
  const msg = isRecord(event.message) ? event.message : null;
  const usage = msg && isRecord(msg.usage) ? msg.usage : null;
  if (!usage) return null;

  const inputTokens = usage.inputTokens ?? usage.input_tokens;
  const contextWindow = usage.contextWindow ?? usage.context_window;
  if (typeof inputTokens !== 'number' || typeof contextWindow !== 'number') return null;

  return { contextTokens: inputTokens, contextWindow };
}
```

Create `src/orcd/pi-runtime.ts`:

```ts
import { createAgentSession, DefaultResourceLoader } from '@earendil-works/pi-coding-agent';
import { getModel } from '@earendil-works/pi-ai';

export interface CreatePiRuntimeSessionOpts {
  cwd: string;
  providerId: string;
  modelId: string;
  effort?: string;
}

export interface PiRuntimeSession {
  id: string;
  prompt(text: string, opts?: { streamingBehavior?: 'steer' | 'followUp' }): Promise<void>;
  subscribe(cb: (event: unknown) => void): void;
  abort(): Promise<void>;
  compact(instructions?: string): Promise<unknown>;
  setEffort(effort: string): Promise<void>;
  getMessages(): unknown[];
}

function effortToThinkingLevel(effort: string | undefined): string {
  if (effort === 'disabled') return 'disabled';
  if (effort === 'low' || effort === 'medium' || effort === 'high' || effort === 'max') return effort;
  return 'high';
}

export async function createPiRuntimeSession(opts: CreatePiRuntimeSessionOpts): Promise<PiRuntimeSession> {
  const resourceLoader = new DefaultResourceLoader({
    cwd: opts.cwd,
    agentDir: `${process.env.HOME ?? ''}/.pi`,
  });
  await resourceLoader.reload();

  const model = getModel(`${opts.providerId}/${opts.modelId}`);
  const session = await createAgentSession({
    cwd: opts.cwd,
    model,
    thinkingLevel: effortToThinkingLevel(opts.effort),
    resourceLoader,
  });

  return {
    id: String(session.sessionId ?? ''),
    prompt: (text, promptOpts) => session.prompt(text, promptOpts),
    subscribe: (cb) => { session.subscribe(cb); },
    abort: () => session.abort(),
    compact: (instructions) => session.compact(instructions),
    setEffort: async (effort) => {
      if (typeof session.setThinkingLevel === 'function') {
        await session.setThinkingLevel(effortToThinkingLevel(effort));
      }
    },
    getMessages: () => Array.isArray(session.messages) ? session.messages : [],
  };
}
```

If Pi's actual `createAgentSession` signature differs, adapt this wrapper only; do not leak Pi constructor details into `session.ts`.

```ts
  const log = (msg: string) => console.log(`[orcd:${this.id.slice(0, 8)}] ${msg}`);

    const piSession = this.activePiSession ?? await createPiRuntimeSession({
      cwd: this.cwd,
      providerId: this.provider,
      modelId: this.model,
      effort: opts.effort,
    });
    this.activePiSession = piSession;

    piSession.subscribe((event) => {
      if (this.state === 'stopped') return;
      const payload = mapPiEventToOrcdPayload(event);
      const eventIndex = this.buffer.push(payload);

      const usage = getContextUsageFromPiEvent(event);
      if (usage) {
        this.lastContextTokens = usage.contextTokens;
        this.lastContextWindow = usage.contextWindow;
      }

      if (typeof payload === 'object' && payload !== null && 'type' in payload && payload.type === 'result') {
        const msg: SessionResultMessage = { type: 'result', sessionId: this.id, eventIndex, result: payload };
        for (const cb of this.subscribers) cb(msg);
      } else {
        const msg: StreamEventMessage = { type: 'stream_event', sessionId: this.id, eventIndex, event: payload };
        for (const cb of this.subscribers) cb(msg);
      }

      if (usage) {
        const cuMsg: ContextUsageMessage = {
          type: 'context_usage',
          sessionId: this.id,
          contextTokens: usage.contextTokens,
          contextWindow: usage.contextWindow,
        };
        for (const cb of this.subscribers) cb(cuMsg);
      }
    });

    log(`started (resume=${!!opts.resume}, model=${this.model})`);
    await piSession.prompt(opts.prompt, opts.resume ? { streamingBehavior: 'followUp' } : undefined);
    if (this.state !== 'stopped') this.state = 'completed';
    log(`exited (state=${this.state})`);
  } catch (err) {
    const errStr = String(err);
    if (errStr.includes('abort') || errStr.includes('AbortError')) {
      this.state = 'stopped';
    } else {
      this.state = 'errored';
      const msg: SessionErrorMessage = { type: 'error', sessionId: this.id, error: errStr };
      for (const cb of this.subscribers) cb(msg);
    }
  } finally {
    await this.runBeforeExitHooks();
    const exitMsg: SessionExitMessage = {
      type: 'session_exit',
      sessionId: this.id,
      state: this.state as 'completed' | 'errored' | 'stopped',
    };
    for (const cb of this.subscribers) cb(exitMsg);
  }
}
```

```ts
export function textFromPiContent(content: unknown): string {
  if (typeof content === 'string') return content;
  if (!Array.isArray(content)) return '';

  const parts: string[] = [];
  for (const block of content) {
    if (typeof block !== 'object' || block === null) continue;
    const rec = block as Record<string, unknown>;
    if (rec.type === 'text' && typeof rec.text === 'string') parts.push(rec.text);
    if (rec.type === 'tool_call' && typeof rec.name === 'string') parts.push(`[tool: ${rec.name}]`);
    if (rec.type === 'tool_result' && typeof rec.text === 'string') parts.push(`[tool result: ${rec.text.slice(0, 500)}]`);
  }
  return parts.join('\n');
}

export function buildMemoryExcerptFromPiMessages(messages: unknown[], maxChars: number): string {
  const out: string[] = [];
  let total = 0;

  for (const msg of messages) {
    if (typeof msg !== 'object' || msg === null) continue;
    const rec = msg as Record<string, unknown>;
    const role = rec.role === 'user' || rec.role === 'assistant' ? rec.role : 'message';
    const text = textFromPiContent(rec.content);
    if (!text.trim()) continue;
    const line = `[${role}]: ${text}`;
    if (total + line.length > maxChars) break;
    out.push(line);
    total += line.length;
  }

  return out.join('\n\n');
}
```
